================================================== ====================== 
     Active Template Library: P2PWebAX Project Overview 
================================================== ====================== 

Application Wizard you created this P2PWebAX project to use as your 
Starting point for writing dynamic link library (DLL) is. 
The project implemented by Visual C ++ property. 

This file contains 
Project summary of each file. 

P2PWebAX.vcproj 
     This is the main project file generated by Application Wizard VC ++ project. 
     It contains information about the generated Visual C ++ version of this document, as well as 
     For information about using the Application Wizard selected 
     Information platform, configuration and project functions. 

_P2PWebAX.idl 
     This file will be generated by the compiler when you build the project. 
     It contains the IDL definition of the type library, interfaces and co-class project definition. 
     This file is processed by the MIDL compiler to generate: 
         C ++ interface definitions and GUID declarations (_P2PWebAX.h) 
         GUID definition (_P2PWebAX_i.c) 
         Type Library (_P2PWebAX.tlb) 
         Marshaling code (_P2PWebAX_p.c and dlldata.c) 
P2PWebAX.cpp 
     This file contains the object map and DLL export realization. 
P2PWebAX.rc 
     This is all the Microsoft Windows resources that the program uses 
     Listing. 

P2PWebAX.def 
     This module definition file provides information necessary for the DLL linker 
     Exported information. It contains the following export: 
         DllGetClassObject 
         DllCanUnloadNow 
         GetProxyDllInfo 
         DllRegisterServer 
         DllUnregisterServer
